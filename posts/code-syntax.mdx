### 2. Line Highlighting

Many Markdown renderers and plugins (especially in static site generators or custom React components) support line highlighting using curly braces after the language:

````javascript
function foo() {
  // Line 2 is highlighted
  console.log("Hello");
  // Lines 4 and 5 are highlighted
  return true;
}
```
